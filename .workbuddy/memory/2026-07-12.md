### 关键发现（踩坑）
- **TaskIn schema 无 `status` 和 `sprint_id` 字段**: 创建 Task 时 status 默认为 "backlog"，sprint_id 默认为 null。改状态需用 `PUT /api/tasks/{tid}/status`，分配 Sprint 需用 `PATCH /api/tasks/{tid}`。
- **Sprint update 用 PATCH 不是 PUT**: `PUT /api/sprints/{sid}` 返回 405 Method Not Allowed。
- **SprintIn 用 `title` 不是 `name`**: 与 ProjectIn 的 `name` 字段不同。
- **任务状态机不允许 backlog → in_progress 直接跳转**: 需走 backlog → todo → in_progress。
- **Playwright 未预装**: 需 `pip install playwright && playwright install chromium`。
