### Angular 布局修复（重要！）
- 问题：Angular 默认使用 Emulated ViewEncapsulation，导致全局 styles.css 无法应用到组件元素
- 症状：侧栏跑到右边、主内容溢出  
- 解决：在 app.ts 的 @Component 中添加 encapsulation: ViewEncapsulation.None
- 部署：修改后需 npm run build 重建，然后 docker cp 部署到容器
